# Corolla Fix Helper — Project Brief + Master Build Checklist

## Project Summary

Corolla Fix Helper is a private, local-first repair document organizer and lookup app built first for a **2009 Toyota Corolla LE 1.8L**. Its job is to help me find the right repair information fast, keep notes tied to real problems, and turn scattered manuals and writeups into a usable repair workflow.

This is **not** an AI mechanic app in version 1.

Version 1 should already be valuable without chat, embeddings, or advanced AI.

## Primary Goal

In under **30 seconds**, I can search for a Corolla issue and get to the most relevant documents, notes, and procedures.

## Target User

Me, while troubleshooting, repairing, researching, or organizing repair knowledge for my Corolla.

## Version 1 Scope

### Must Have

* Document import
* Metadata editing
* Search
* Filters
* Favorites
* Notes
* Symptoms
* Procedures
* Clean dashboard
* Local storage
* SQLite database

### Not in Version 1

* AI chat over all documents
* Embeddings/vector database
* OCR pipeline for everything
* Multi-vehicle support
* Cloud sync
* User accounts
* Voice assistant
* Image-based diagnosis
* Parts store integrations

## Product Rules

1. Start with one vehicle only.
2. Documents first, AI second.
3. Real usefulness beats flashy features.
4. Do not rebuild from scratch unless absolutely necessary.
5. Import files in batches, not all at once.

## Recommended Stack

* Frontend: React + Vite + Tailwind
* Backend: Node.js + Express
* Database: SQLite
* Storage: Local uploads folder + extracted text in database or cache files

## Core App Sections

* Dashboard
* Documents
* Search
* Symptoms
* Procedures
* Notes
* Settings

## First Real Dataset

Start with **25–75 high-value files**, not all 2,000.

Prioritize:

* Belt / pulley / tension docs
* Battery / charging / cable connection docs
* Radiator / cooling system docs
* A/C service docs
* Wiring diagrams
* Common maintenance procedures

## Milestone Definition

The first real milestone is reached when:

* I can import a PDF
* edit its metadata
* search for it by keyword
* filter results by system/type
* open the file quickly
* link it to a symptom or procedure
